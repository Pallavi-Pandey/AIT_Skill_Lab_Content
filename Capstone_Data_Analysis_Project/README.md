# Capstone Data Analysis Project: HR Analytics

Welcome to the Final Capstone project! This project serves as a comprehensive culmination of everything you've learned in the Data Analytics course, using a real-world dataset directly from Kaggle.

## Objectives
- Perform comprehensive Data Cleaning and handling of Missing/Inconsistent records.
- Conduct Exploratory Data Analysis (EDA) on an Employee Attrition & Performance dataset.
- Build multi-layered visual insights that answer critical business questions.
- Formulate data-driven recommendations.

## The Dataset
We are utilizing the **IBM HR Analytics Employee Attrition & Performance** dataset, widely used by analysts. The dataset features information on employee satisfaction, income, work-life balance, and whether the employee left the company (attrition).

- **Source:** [Kaggle - IBM HR Analytics](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset)
- **Local Location:** The dataset is already downloaded and available in `data/WA_Fn-UseC_-HR-Employee-Attrition.csv`.
