## Key Skills Gained

- Write SQL queries using SELECT, WHERE, GROUP BY, HAVING, and JOINs
- Define and calculate business KPIs (Revenue, ARPU, Churn Rate, CLV)
- Identify sales trends and customer segments from structured data
- Structure a business presentation using the Minto Pyramid (Situation → Complication → Resolution)
- Translate a chart into a one-sentence actionable business recommendation
