## Key Skills Gained

- Calculate and interpret mean, median, standard deviation, and IQR
- Identify and handle outliers using the IQR method
- Read a correlation matrix to find variable relationships
- Select the correct chart type for each data scenario
- Spot and avoid misleading visualization practices
- Build multi-panel dashboards with Matplotlib
